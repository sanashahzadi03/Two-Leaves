# Two-Leaves

Two Leaves and a Bud — 1:1 Pixel-Perfect Frontend Clone

A strict, pixel-perfect frontend clone of the Two Leaves and a Bud organic tea and matcha e-commerce homepage. This project focuses on high-fidelity visual accuracy, clean semantic layouts, responsive scalability, and custom UI design patterns using pure HTML5 and Vanilla CSS.

Key Features
Pixel-Perfect Accuracy: Closely replicates the brand typography, exact spacing guidelines, layout bounds, and visual aesthetics of the original site.

Sticky Floating Navbar & Brand Wrapper: An adaptive navigation wrapper that cleanly scales from transparent hero overlay down to a tight, border-styled floating bar upon scroll.

Advanced Desktop Mega Menus: Completely custom CSS multi-column dropdown trays displaying categories, varieties, collections, and custom styled product promo cards.

Fluid Responsive Layouts: Optimized for all viewport sizes (desktop down to mobile) using native media queries, flexible boxes, and CSS Grid layouts.

Fully Formatted UI Components: Includes high-fidelity global elements like custom announcement bars, specific call-to-action button systems, badge counts, and dynamic graphic sections.

Design & Technical Implementation
Semantic Structure: Built completely using clean, accessible, and structured semantic web elements (<header>, <nav>, <main>, <section>, <picture>).

Color Palette: Standardized on brand-specific tones such as Primary Indigo (#2a2857), Organic Cream (#fbf9eb), and Deep Charcoal (#201c1a).

Typography System: Implemented unique font pairings combining modern body elements (Figtree, Inter) with stylized custom display headings (RecklessCondensed and MidnightSans).

Custom Vector Work: Houses exact embedded brand SVG icons for interactive cart triggers, promotional graphics, and navigation identifiers.
